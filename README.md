## Описание / Инструкция по выполнению задания:

### Родительский класс `Animal`
Свойства класса:
- **name** (`String`)
- **age** (`int`)
- **weight** (`double`)
- **color** (`String`)

Методы класса:
- **Getters & Setters**
- **`say()`**: выводит сообщение `"Я говорю"`
- **`go()`**: выводит сообщение `"Я иду"`
- **`drink()`**: выводит сообщение `"Я пью"`
- **`eat()`**: выводит сообщение `"Я ем"`

#### Метод `toString()`
Переопределенный метод возвращает строку вида:
Привет! Меня зовут {name}, мне {age} лет(/год/года), я вешу - {weight} кг, мой цвет - {color}

Количество лет определяется следующим образом:
- 1 → *«год»*
- 2, 3, 4 → *«года»*
- Остальные случаи → *«лет»*

Пример вывода:
Привет! Меня зовут Бобик, мне 5 лет, я вешу - 15 кг, мой цвет - черный.

### Основная программа
Используется коллекция **ArrayList\<Animal\>**. Меню создается через консоль с командами: **add**, **list**, **exit**.

- Команда **`add`**: запрашивает название животного (**cat**, **dog**, **duck**) и просит ввести имя, возраст, вес и цвет. После инициализации экземпляра вызывает метод **`say()`**.
- Команда **`list`**: выводит на экран результат метода **`toString()`** для каждого элемента коллекции.
- Команда **`exit`**: выход из приложения.

Команды оформлены в виде **enum** и нечувствительны к регистру, лишняя пунктуация удаляется.
